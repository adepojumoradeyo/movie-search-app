# Movie Search App

Author: Adepoju Ebunlomo

## Description
A movie search application built with Vanilla JavaScript using the OMDb API.

## How to Get API Key
1. Visit http://www.omdbapi.com/apikey.aspx
2. Sign up for free key

## How to Run Locally
1. Clone repository
2. Open index.html in browser

## Live Demo
https://movie-search-app-wheat-two.vercel.app/

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- OMDb API

## Features
- Movie search
- Clickable movie cards
- Detailed view
- Loading state
- Error handling
- Responsive design
